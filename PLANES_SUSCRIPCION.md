## ⚙️ Configuración de Features

Para asignar features a los planes que aún no las tienen configuradas (`free_trial`, `basic`, `professional`), ejecutar en Supabase SQL Editor:

```sql
-- Plan Gratuito Permanente (features básicas)
INSERT INTO subscription_plan_features (subscription_plan_id, feature_code)
SELECT sp.id, feature_code
FROM subscription_plans sp
CROSS JOIN (
    VALUES ('dashboard'), ('pos'), ('cash_close'), ('catalog'), ('crm')
) AS features(feature_code)
WHERE sp.code = 'free_trial'
ON CONFLICT DO NOTHING;

-- Plan Básico
INSERT INTO subscription_plan_features (subscription_plan_id, feature_code)
SELECT sp.id, feature_code
FROM subscription_plans sp
CROSS JOIN (
    VALUES 
        ('dashboard'), ('pos'), ('cash_close'),
        ('inventory'), ('catalog'), ('team'),
        ('crm'), ('reports')
) AS features(feature_code)
WHERE sp.code = 'basic'
ON CONFLICT DO NOTHING;

-- Plan Profesional
INSERT INTO subscription_plan_features (subscription_plan_id, feature_code)
SELECT sp.id, feature_code
FROM subscription_plans sp
CROSS JOIN (
    VALUES 
        ('dashboard'), ('pos'), ('cash_close'),
        ('reception'), ('milling'), ('roasting'),
        ('laboratory'), ('packaging'),
        ('finance'), ('inventory'), ('catalog'),
        ('team'), ('crm'), ('suppliers'), ('reports'), ('promotions')
) AS features(feature_code)
WHERE sp.code = 'professional'
ON CONFLICT DO NOTHING;

-- Plan Empresarial (TODAS las features)
INSERT INTO subscription_plan_features (subscription_plan_id, feature_code)

        ('projections'), ('team'), ('crm'),
```

## ⏰ Gestión del Trial

### Asignación Automática
- ✅ Nuevas organizaciones reciben automáticamente el **Trial de 14 días**
- ✅ Acceso completo a todas las funcionalidades
- ✅ No requiere tarjeta de crédito

### Después del Trial
1. **Opción 1:** Elegir un plan de pago (Básico, Profesional, Empresarial)
2. **Opción 2:** Cambiar al Plan Gratuito permanente (limitado)
3. **Opción 3:** Perder acceso hasta seleccionar un plan

### Verificar Estado del Trial
```sql
SELECT 
    name,
    trial_ends_at,
    CASE 
        WHEN trial_ends_at > NOW() THEN 'Activo'
        WHEN trial_ends_at <= NOW() THEN 'Expirado'
        ELSE 'Sin trial'
    END as estado_trial,
    trial_ends_at - NOW() as tiempo_restante
FROM organizations
WHERE subscription_plan_id IN (
    SELECT id FROM subscription_plans WHERE code = 'trial'
);
```
